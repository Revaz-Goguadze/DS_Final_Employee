# Project Contributions

## Team Structure
This is a solo project submitted by the student.

## Roles & Responsibilities
- **Data Acquisition & Cleaning**: Responsible for sourcing the IBM HR Analytics dataset, handling missing values, and preparing the data for analysis.
- **Exploratory Data Analysis (EDA)**: Created visualizations to understand attribute distributions and correlations with attrition.
- **Machine Learning Modeling**: Implemented Logistic Regression, Random Forest, Gradient Boosting, and Decision Tree models, including hyperparameter tuning and evaluation.
- **Reporting**: Generated final reports, analysis documents, and presentation assets.

## Timeline

- **Week 2**: EDA and Feature Engineering.
